# aoc_twenty

A Gleam program

## Quick start

```sh
# Build the project
rebar3 compile

# Run the eunit tests
rebar3 eunit

# Run the Erlang REPL
rebar3 shell
```

## Execute A Run

Print the output for both parts of a given day:

```sh
# Build the escript
rebar3 escriptize

# Run for a specific day
_build/default/bin/aoc_twenty <day>
```
